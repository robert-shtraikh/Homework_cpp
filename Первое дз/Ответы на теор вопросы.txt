1)Создатель языка - Бьёрн Страуструп. Высокоэффективный язык C был усовершенствован добавлением 
возможности работы с классами. Позже в 1985 году был переименован в C++.
Основные характеристики:
	1. Низкоуровневый
	2. Сложный
	3. Компилируемый
	4. Мультипарадигмальный
	5. Статически типизированный
	6. Эффективный

2)	1. Логический (bool) - принимает значение True/False;
	2. Символьный (char) - хранит один из символов ISO-646;
	3. Целочисленный (int) - целые числа;
	4. Типы с плавающей точкой (double, float) - дроби;
	5. void - указание отсутствия информации;
	6. Перечислимые типы (enum) - для представления значений из конкретного множества

3) Под переносимостью прикладной программы (ПП) понимается возможность использования ПП на различных аппаратных и операционных платформах, отличающихся по архитектуре и характеристикам, с сохранением 
или небольшим изменением функций ПП. Например, на одну и ту же переменную типа integer может быть выделено разное количество байт в зависимости от устройства.
Проблема решается, если не использовать тип int для больших чисел.

4)	1. Определение - определение сущности (реализация типа), которая соответствует идентификатору или выделение памяти объекту.
	2. Объявление - введение идентификатора и описание типа.
	3. Инициализация - одновременное объявление и присваивание значения переменной.
	4. Присваивание - механизм связывания имени объекта (переменной) с его значением.

5) Центральные процессоры в качестве основной единицы работы с памятью используют мащинное слово, размер которого может быть
различным.
Как правило машинное слово равно 2^k байтам.
При сохранении объекта в памяти может случиться, что некое поле, состоящее из некольких байтов,
пересечёт "естественную границу" слов в памяти.
Некоторые модели процессоров не могут обращаться к данным в памяти, нарушающим границы машинных слов.
Некоторые могут обращаться к невыровненным данным дольше, нежели к данным, находящимся внутри целого машинного слова.
